# Excel Sheet Organizer 1.0.0

This organizer takes the Datamade legislators csv and converts it to a single organized excel file. 

##Installation

The python file can be ran from the command prompt or any editor. Make sure the python file runs in the same directory as the legislators csv. No local pathways were used to simplify third party access. 

##Usage

In your command line run the code below or simply run in the python script in your editor of choice. 

	python DataMade.py
